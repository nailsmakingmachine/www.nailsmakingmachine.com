# Nails Making Machine - Official Website

Welcome to the official website repository for **Nails Making Machine**, a leading manufacturer of high-speed automatic nail making and binding wire machinery based in Rajkot, Gujarat. 

This website serves as the digital front door for our brand, **Nails Making Machine**, showcasing our premium industrial engineering to a global market.

## 🚀 Features

* **Ultra-Premium Design:** Built with a modern, dark-themed industrial aesthetic using Tailwind CSS.
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices.
* **Master Catalog:** A dedicated machinery fleet page showcasing all automatic models.
* **Fast Performance:** Lightweight HTML and CSS for lightning-fast loading speeds.
* **Direct Communication:** Integrated WhatsApp quick-links for immediate price quotes and inquiries.

## 📁 Repository Structure

This website is structured using separate, clean HTML files for maximum SEO performance and organization:

* `index.html` - The main landing page.
* `machinery.html` - The master catalog grid showcasing the entire machine fleet.
* `manufacturer.html` - Information about Indian Wire Product and our manufacturing vision.
* `aboutus.html` - Deep dive into our mission and engineering standards.
* `location.html` - Maps and factory address.
* `contact.html` - Direct contact details for our global desk.
* `in1.html` - Dedicated product page for the IN1+ Model (600 pcs/min).
* `in3.html` - Dedicated product page for the IN3 Model (350 pcs/min).
* `in4.html` - Dedicated product page for the IN4 Model (250 pcs/min - Best Seller).
* `in6.html` - Dedicated product page for the IN6+ Model (150 pcs/min).
* `ihs90.html` - Dedicated product page for the IHS90 High-Speed Model (700 pcs/min).
* `wiredraw.html` - Dedicated product page for our high-tensile Wire Drawing machinery.

## 🛠️ Technology Stack

* **HTML5:** Semantic markup for robust SEO.
* **Tailwind CSS:** Utility-first CSS framework for rapid, highly customized styling directly within the HTML.
* **Google Fonts:** Utilizing 'Syncopate' and 'Inter' for premium typography.
* **Unsplash API:** High-resolution placeholder imagery for industrial equipment.

## 🌐 How to Deploy (GitHub Pages)

This website is designed to be hosted for free using GitHub Pages.

1. Upload all `.html` files to your GitHub repository (named `yourusername.github.io`).
2. Go to your repository **Settings**.
3. Navigate to **Pages** on the left sidebar.
4. Ensure the source is set to `main` (or `master`) branch and `/ (root)` folder.
5. Save, and your website will be live in a few minutes!
6. (Optional) Enter your custom domain name in the "Custom domain" section to route your live URL.
